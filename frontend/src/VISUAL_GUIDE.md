### Обратите внимание:
```
📦 ПО КОЛИЧЕСТВУ:           💰 ПО ПРИБЫЛИ:
- Главная цифра: штуки      - Главная цифра: сумма
- Дополнительно: выручка    - Дополнительно: штуки
- Порядок: по количеству    - Порядок: по деньгам
```

## 📊 СРАВНЕНИЕ:

### Рейтинг товаров:

```
┌────────────────────────┬────────────────────────┐
│   📦 ПО КОЛИЧЕСТВУ     │    💰 ПО ПРИБЫЛИ       │
├────────────────────────┼────────────────────────┤
│ #1 Дешевый товар       │ #1 Дорогой товар       │
│    (много продаж)      │    (много денег)       │
│                        │                        │
│ #2 Средний товар       │ #2 Средний товар       │
│                        │                        │
│ #3 Дорогой товар       │ #3 Дешевый товар       │
│    (мало продаж)       │    (мало денег)        │
└────────────────────────┴────────────────────────┘

Пример:
┌────────────────────────┬────────────────────────┐
│ #1 Чехол для телефона  │ #1 iPhone 14 Pro       │
│    500 продаж × 5,000  │    20 продаж × 500,000 │
│    = 2,500,000 сум     │    = 10,000,000 сум    │
└────────────────────────┴────────────────────────┘
```

## 🎯 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:

### Пример 1: Компания хочет знать что популярно

```
1. Открыть "Расширенная аналитика"
2. Нажать "📦 По количеству"
3. Увидеть:
   #1 AirPods - 200 продаж
   #2 Чехол - 150 продаж
   #3 iPhone - 50 продаж
   
Вывод: AirPods популярнее всего!
```

### Пример 2: Компания хочет знать что прибыльно

```
1. Открыть "Расширенная аналитика"
2. Нажать "💰 По прибыли"
3. Увидеть:
   #1 iPhone - 25,000,000 сум
   #2 MacBook - 15,000,000 сум
   #3 AirPods - 8,000,000 сум
   
Вывод: iPhone приносит больше всего денег!
```

### Пример 3: Покупатель звонит

```
Покупатель: "Я вчера купил что-то, где мой заказ?"

Компания открывает историю платежей:
┌──────────┬─────────────────────────┐
│ 06.01    │ • iPhone 14 Pro ×1      │
│ 15:30    │ • Чехол силиконовый ×2  │
│ Иван     │                         │
└──────────┴─────────────────────────┘

Компания: "Вы купили iPhone и 2 чехла вчера в 15:30"
Покупатель: "Да, точно! Когда получу?"

✅ Быстрый ответ без кликов по деталям!
```

## 💡 СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ:

### Для увеличения продаж:
```
1. Смотрите "📦 По количеству"
2. Найдите популярные товары
3. Закажите их больше на склад
4. Сделайте специальное предложение
```

### Для увеличения прибыли:
```
1. Смотрите "💰 По прибыли"
2. Найдите прибыльные товары
3. Сделайте рекламу для них
4. Предлагайте их покупателям
```

### Для лучшего сервиса:
```
1. Откройте "История платежей"
2. Сразу видите что купил покупатель
3. Быстро отвечаете на вопросы
4. Покупатель доволен!
```
